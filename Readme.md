# node-thetav

Node.js Library for Richo Theta V

## Install

```
npm install --save node-thetav
```

## Require ##

```javascript
import thetav from 'node-thetav';
const httpApi = thetav.http;
const bleApi = thetav.ble;
```

## Usage

import thetav from 'node-thetav';
